# 머신러닝(ML) 중요 개념의 이해

머신러닝 전체 워크플로우와 핵심 개념을 정리한 학습 노트입니다.

---

## 핵심 요약

### 1. ML의 위치

`ML → DL → LLM → Agent`로 이어지는 흐름의 시작점. 정해진 절차(데이터 준비 → 피처 엔지니어링 → 모델 학습 → 평가 → 최적화 → 서빙)를 따른다.

### 2. 학습 방법

| 방법 | 세부 분류 |
|---|---|
| 지도 학습 | 분류, 회귀 |
| 비지도 학습 | 군집, 차원 축소 |
| 강화 학습 | 유한게임, 시뮬레이션 환경(Unity ML-Agents 등) |

### 3. ML 모델 학습 절차 (10단계)

1. **사전 준비** — 미션·데이터 이해
2. **베이스라인 구축** — 소량 데이터로 끝까지 한 번
3. **알고리즘 선택** — 지도/비지도, 베이스 모델 → 앙상블(부스팅)
4. **EDA & 피처 엔지니어링** — 스케일링, 인코딩, 전처리
5. **학습** — 교차 검증(K-Fold, 층화 K-Fold)
6. **예측** — 미학습 데이터로 검증
7. **성능 평가** — 분류(정확도·F1·AUC), 회귀(MSE·RMSE), 군집(실루엣 계수)
8. **최적화·반복** — 그리드/랜덤/베이지안 서치
9. **모델 덤프** — joblib → S3
10. **모델 서빙/MLOps** — 엔드포인트, MLFlow

### 4. 핵심 인사이트

> 모델 선택·최적화·블렌딩은 AutoML이 담당. 결국 경쟁의 핵심은 **양질의 데이터와 피처 엔지니어링**이다.
