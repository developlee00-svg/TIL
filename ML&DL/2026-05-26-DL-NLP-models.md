# NLP 모델 — RNN부터 LLM까지

자연어 처리의 신경망 진화 흐름(RNN → LSTM → GRU → 트랜스포머 → LLM)과 생성형 모델 계보를 정리한 학습 노트입니다.

---

## 핵심 요약

### 1. NLP의 본질

순서가 있는 데이터(자연어)를 다루는 분야. 단어 하나만 보면 의미가 없고, 앞뒤 맥락 속에서만 의미가 성립 → 일반 DNN으로는 부족, 별도 신경망 구조 필요.

### 2. RNN 계열 진화

| 모델 | 핵심 아이디어 | 단점 |
| --- | --- | --- |
| RNN | 이전 은닉층 → 다음 은닉층으로 데이터 전달 | 장기 기억 소실, 기울기 폭주/소실 |
| LSTM | 장기기억선(Cell state) + 게이트 3개 | 파라미터 증가, 비용 상승 |
| GRU | 장기기억선 제거, 게이트 2개로 경량화 | 데이터 클 땐 LSTM이 우수 |

### 3. LSTM 핵심 게이트

- **Ct (Cell state)** — 장기기억선, 중요 기억 보관
- **Ft (Forget)** — 잊혀질 데이터 삭제
- **It (Input)** — 신규 입력 중 중요 정보 선별
- **Ot (Output)** — 현재 시점 은닉 상태값 생성

### 4. 입출력 구조와 활용

- **many-to-many** — 기계번역(비대칭), 챗봇·QA(대칭)
- **many-to-one** — 감정분석, 요약·키워드 추출
- **one-to-many** — 이미지 캡셔닝, 글쓰기

### 5. 어텐션 메커니즘 (2017)

> "Attention is All You Need" — RNN 패러다임을 무너뜨린 결정적 논문.

- **핵심** — 토큰 단위 가중치 부여, 예측 단어와 연관된 단어에 집중
- **전제 조건** — 인코더-디코더 구조 + 대량 데이터 학습 필수
- RNN의 "순서대로 누적 기억" → 어텐션의 "전체를 보고 어디 집중할지 결정"

### 6. 트랜스포머

- 순환 구조 폐기 → **모든 토큰 동시 처리 가능 → 병렬화 → GPU 활용 극대화**
- 대규모 학습 가능 → LLM 시대를 연 결정적 기술 변화
- 허깅페이스 생태계 표준: **토크나이저(자연어 → 토큰) + 모델(토큰 → 처리)**

### 7. 트랜스포머 계열 분기

| 계열 | 대표 모델 | 특화 | 개발 주체 |
| --- | --- | --- | --- |
| 인코더-디코더 | BART | 변환(번역·요약) | Meta |
| 인코더 | BERT, ELECTRA | 이해(분류·QA) | Google |
| 디코더 | GPT | 생성 | OpenAI |

> 현재 LLM 흐름은 **디코더 기반(GPT 계열)** 이 주도. 2022년 말 ChatGPT 등장 = 특이점.

### 8. 생성형 모델 계보

```
판별 모델 : X → Y 확률 예측
생성 모델 : 재료 → X 생성 (이미지·영상·텍스트·사운드)
  └─ 딥러닝 생성 모델
       ├─ Explicit: AE, VAE, U-Net
       ├─ Implicit: GAN (DCGAN, StyleGAN)
       ├─ Flow Model
       └─ Diffusion Model ⭐ 현재 주류
              (DALL-E, Stable Diffusion, NANOBANANA, D-ID)
```

### 9. GAN의 적대적 학습

- **생성자(Fake Maker)** — 가짜 데이터 생성
- **판별자(Fake Finder)** — 가짜 탐지
- 위조지폐범 vs 경찰 비유 — 서로 경쟁하며 양쪽 모두 발전

### 10. 핵심 인사이트

> 자연어 처리의 역사는 **"기억을 어떻게 다룰 것인가"** 의 역사다. RNN은 순환으로, LSTM은 게이트로, 트랜스포머는 어텐션으로 답을 내놓았고, 그 결과 LLM에 도달. 회귀의 w → CNN 커널 → 어텐션 가중치는 모두 같은 원리(중요한 것에 가중치 부여 + 학습으로 최적화)를 다른 도메인에 적용한 것.
